# Mein Portfolio
Das ist meine persönliche Portfolio-Webseite mit einem integrierten Kontaktformular

## Funktionen
* **Responsive Design** optimiert für verschiedene Bildschirmgrößen
* **Kontaktformular** einfache Möglichkeit zur Kontaktaufnahme (PHP-Script läuft auf einem ecternen Webhoster)
* **Moderner UI** erstellt mit HTML5, CSS3, JavaScript sowie JSON-Dateien

## Installation & Start
um dieses Projekt lokal auf Ihrem Computer anzuschauen:

1. Klonen Sie das Repository:
   ```bash
   git clone https://github.com/lindawitznick/my-portfolio.git
   ```
2. Öffnen Sie die Datei `index.html` direkt in Ihrem Webbrowser.

## Wichtige Hinweise
* **Hosting** dieses Repository dient nur zur Code-Verwaltung. Die Live-Webseite wird auf einem externen Webserver verwaltet, auf den die Dateien direkt via MobaXterm hochgeladen wurden.
* **Sicherheit** sensible Zugangsdaten, Passwörter und Server-Konfigurationen wurden aus Sicherheitsgründen absichtlich nicht in dieses öffentliche GitHub-Repository hochgeladen.
